This directory contains all the project files for our game 'Mayhem Island'
The built executable version can be found in Builds > Windows > x86 > MayhemIsland.exe
Also included in this directory is a document (3rd Party Assets Used.docx )listing all of the assets 
	we outsourced, either from the in-engine Asset Store or from other sources online.
Trailer.mp4 is a short audiovisual that we produced to serve as a trailer for the game, used to 
	introduce it during our final demonstration.
